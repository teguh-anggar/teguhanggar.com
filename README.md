## 🚀 How to Run Locally

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git for version control

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/teguhanggar/personal-website.git
   cd personal-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in browser**
   Navigate to `http://localhost:3000` to view the website

### Build for Production
```bash
npm run build
# or
yarn build
```
